# PWA Icons

Place the following PNG files in this directory for the PWA manifest to work:

| File | Size | Notes |
|------|------|-------|
| `pwa-192x192.png` | 192 × 192 px | Used by Android / Chrome |
| `pwa-512x512.png` | 512 × 512 px | Used for splash screens & maskable icon |

## Design guidelines

- Use the Centre Alilham logo, or the letter **"A"** in white on a `#7C3AED` (soft-600) background.
- For the maskable icon, keep the logo centered and within the inner 80 % of the canvas (safe zone).
- Export as PNG with a transparent or white background.

## Quick generation

You can generate these icons from any source image using tools such as:
- [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net)
- [Squoosh](https://squoosh.app) for manual resizing
